Создай каркас приложения доставки пиццы в файле src/index.html.

Всё должно быть в отдельных файлах: HTML, CSS, JS.
Подключи style.css и script.js в index.html.

Добавь функционал корзины в приложение доставки пиццы.

ЗАДАЧА 1: Логика добавления в корзину

В файле src/script.js:

1. Создай массив cart = [] для хранения товаров
2. При клике на кнопку "Добавить в корзину":
   - Получить данные пиццы (название, цена, id)
   - Проверить, есть ли уже эта пицца в корзине
   - Если есть → увеличить количество на 1
   - Если нет → добавить новый товар с количеством = 1
   - Сохранить корзину в LocalStorage: localStorage.setItem('cart', JSON.stringify(cart))
   - Обновить счётчик корзины на кнопке

3. Функция обновления счётчика:
   - Посчитать общее количество товаров в корзине
   - Отобразить число на кнопке "Корзина (число)"

4. При загрузке страницы:
   - Прочитать корзину из LocalStorage: JSON.parse(localStorage.getItem('cart'))
   - Если есть сохранённые товары → загрузить их в массив cart
   - Обновить счётчик

ЗАДАЧА 2: Изменить alert на уведомление

Вместо alert("Пицца добавлена!") показать красивое всплывающее уведомление:
- Текст: "✓ Добавлено в корзину"
- Появляется вверху экрана
- Исчезает через 2 секунды
- Зелёный фон

ЗАДАЧА 3: Обновить HTML

В файле src/index.html:
- У каждой кнопки "Добавить в корзину" добавить атрибут data-id с уникальным ID пиццы (1, 2, 3, 4, 5, 6)
- У кнопки "Корзина" добавить span для счётчика: <span id="cart-count">0</span>

Используй vanilla JavaScript, без фреймворков.
