## Создание каркаса приложения
Создай каркас приложения доставки пиццы в файле src/index.html.

Требования:

HTML структура:
- Заголовок "Доставка пиццы 🍕"
- Каталог из 6 пицц в виде сетки (2 колонки)
- Каждая пицца: изображение (placeholder 300x300), название, описание, цена, кнопка "Добавить в корзину"

Пиццы (данные):
1. Маргарита - Томатный соус, моцарелла, базилик - 450₽
2. Пепперони - Томатный соус, моцарелла, пепперони - 550₽
3. Четыре сыра - Моцарелла, пармезан, дор блю, чеддер - 600₽
4. Гавайская - Томатный соус, моцарелла, ветчина, ананасы - 500₽
5. Мясная - Томатный соус, моцарелла, бекон, ветчина, курица - 650₽
6. Вегетарианская - Томатный соус, моцарелла, овощи - 480₽

CSS (в src/style.css):
- Современный дизайн
- Цветовая схема: красный (#E63946) для кнопок, белый фон
- Карточки пицц: белый фон, тень, закруглённые углы
- Адаптивная сетка (2 колонки на десктопе, 1 на мобильном)
- Кнопки с эффектом hover

JavaScript (в src/script.js):
- При клике на "Добавить в корзину" → показать alert "Пицца добавлена!"
- Пока только alert, без реальной логики корзины

Плавающая кнопка "Корзина" внизу экрана (пока без функционала).

Всё должно быть в отдельных файлах: HTML, CSS, JS.
Подключи style.css и script.js в index.html.
